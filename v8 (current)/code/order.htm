Options regarding the layout and structure of your website.

Layout Width

A flexible layout will automatically resize to fit the user's browser size. Fixed layouts stay the same regardless of the size of the browser.

Vertical Alignment

Markup

Advanced options regarding the HTML markup of your website.

Optimization

If you choose documentation, the code will be organized and explained through the use of comments. If you choose speed, all comments will be removed and the code will be compressed, providing a faster load time.

Doctype

If you wish, you can specify a certain doctype for your site to be coded in.

PHP SSI Templating

Implementing PHP SSI templating means that your website will be divided into logical parts (like header, footer, etc). These individual files will be fetched from the server using PHP includes. This makes it easier to make site-wide changes, because you only have to edit one file.

CSS

Advanced options regarding the CSS coding of your website.

Reset Stylesheet

Using a reset stylesheet helps ensure that your website will look the same across all browsers.

Print Stylesheet

Add an additional stylesheet to be used when printing. Only select this if it is likely that your website will be printed.

w3 Valid CSS

This means that the main stylesheet will be valid according to w3 standads. However, if any non-standard browser workarounds are required, they will be placed in a seperate stylesheet which may not validate.

No Inline Styles

This garuntees that all styles will be stored in an external stylesheet. Even if this is not selected, the bulk of the CSS will still be in an external file.

Javascript

Advanced options regarding the Javascript coding of your website.

Javascript Frameworks

If you prefer a specific javascript framework, or prefer that no framework is used, specify it here.

Unobtrusive Javascript

This means that javascript effects will be applied as an enhancement to the user experience, and if a user does not have javascript, their experience will degrade gracefully and still be fully functional.

Drop-Down Menu
