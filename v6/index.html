Welcome

This site contains my experiments in web design and basically acts as a sandbox and repository for all my projects. I'm an amateur web designer, and I do web design as a hobby. This is my first site, and this is the site that carried me through my "noob" years of web design: the nested-tables, the frames, the several hundred markup errors. Now that my web design skills are a bit more mature, I usually work with javascript and php, and post whatever projects I come up with on the site. Feel free to poke around, and if you've got any feedback at all, I'd love to hear it!
